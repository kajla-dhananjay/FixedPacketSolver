\doxysubsection{Detailed Description}
\mbox{\hyperlink{classChannel}{Channel}} defines the control part of the program. 

\mbox{\hyperlink{classChannel}{Channel}} maintains a queue and threads push their updates to it. Once updates from all chains are pushed, it processes those updates.
